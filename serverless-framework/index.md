# Serverless Framework cheat-sheet

## INSTALL
```bash
npm install -g serverless
```

## CREATE PROJECT
```bash
serverless create --template aws-nodejs-typescript --path <folder-name>
```

## INSTALL PLUGIN
```bash
npm install <plugin-name> --save-dev
```

## DEPLOY PROJECT
```bash
sls deploy -v
```
